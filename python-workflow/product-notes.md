# Product Notes: Python Workflow

This guide is a positioning guide, not a Python tutorial.

Useful future product hooks:

- notebook-to-module extraction workflows assisted by Codex
- better visible environment summaries for Python projects
- first-class setup notes generated from package installs
- paper/report integration examples using Python figures and tables
- course templates that combine notebooks, scripts, tests, and RootFS images

Claims to keep careful:

- PythonTeX/PyTeX-style workflows depend on the project being configured with
  those tools; do not imply a special built-in PyTeX product surface unless that
  becomes true.
- Multiple Python versions are supported via normal Linux tools such as `uv`,
  `conda`, or similar tooling, not a separate CoCalc-specific Python version
  manager.
