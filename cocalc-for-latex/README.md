# CoCalc for LaTeX Users

A comparison and positioning guide for LaTeX users who know Overleaf, local TeX
editors, VS Code, LyX, AUCTeX/VimTeX, Tectonic, Typst, and similar writing
tools.

The guide is intentionally respectful: it explains each neighboring product by
its center of gravity, then describes CoCalc as a durable computational project
workspace where the LaTeX paper lives beside the code, notebooks, terminals,
history, collaborators, and agents that support it.
