# CoCalc for Jupyter Users

A comparison and positioning guide for notebook users who already understand
tools such as JupyterLab, VS Code notebooks, Google Colab, Kaggle, Deepnote,
Hex, SageMaker Studio, Databricks notebooks, and marimo.

The guide is intentionally non-hostile: it explains each neighboring product by
its center of gravity, then describes CoCalc as a durable collaborative project
workspace where notebooks are one important document type.
