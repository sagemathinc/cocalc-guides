## Core framing

CoCalc should not be described as a generic LMS replacement. Canvas, Moodle,
Blackboard, and similar systems are strong institution-facing course shells.
CoCalc is strongest when the course work itself happens in live computational
projects: notebooks, terminals, files, LaTeX, data, chat, history, rootfs
environments, and instructor help.
