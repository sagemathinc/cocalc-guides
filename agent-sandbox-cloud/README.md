# CoCalc for Agent Sandboxes

A positioning guide for CoCalc-AI as a durable collaborative project cloud for
humans and agents.

The guide distinguishes CoCalc from public app hosting, raw compute rental,
editor-first cloud development, and disposable execution APIs. It emphasizes
projects, managed dedicated hosts, spot and egress controls, RootFS images,
snapshots, backups, real SSH, browser collaboration, and self-hosting through
Launchpad and Rocket.
