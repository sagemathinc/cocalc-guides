# Using Codex Agent Chat in CoCalc-AI

A short illustrated guide to CoCalc-AI's Codex chat integration.

Audience: project teams, researchers, instructors, and software engineers who
want an agent thread that lives with the project instead of inside a disposable
browser session.

The guide emphasizes:

- durable turns across browser refreshes and project-host interruptions
- rich Markdown/WYSIWYG prompts with images, code blocks, and quotes
- collaborative chatrooms, terminals, notebooks, and project files
- long searchable histories with backend offload
- scheduled Codex runs
- project-native access through the CoCalc CLI and built-in skill
- the boundary between CoCalc's integration and standalone Codex apps
